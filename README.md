Instrucciones para ejecutar el sistema
requisitos:
    Tener instalado Node.js
    Tener instalado y en ejecucion mongodb

1. dirijase a traves de la consola de comandos a la carpeta del Proyecto
2. ejecute la siguiente linea
    "node server.js"
3. momentos despues debera entrar al siguiente enlace desde su navegador
    http://localhost:3000
